# DarkSoul

## How To Install

1. pkg update

2. pkg upgrade

3. pkg install git

4. pkg install Python

4. pkg install Python2

6. pip install requests

7. pip2 install requests

8. pip2 install mechanize

9. git clone https://github.com/DarkSoulsS/Darksoul

10. cd Darksoul

11. python DarkSoul.py
